### Utilisation du Makefile


1/	Compiler le programme:      make

2/	executer le programe :      ./mst <population>  
	exemple:
				./mst 250000


3/	Dessiner le tracé le plus optimise sur un graphique :  make draw 
 	note: Vous devez avoir utiliser le programme 'mst' *avec succes* avant de faire cette commande

 4/	Supprimer tous les fichiers objets (.o), le programme 'mst', le fichier 'result.png' et tous les fichiers output : make fclean
